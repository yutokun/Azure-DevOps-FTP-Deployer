# Azure DevOps FTP Deployer

Deployer script for Azure DevOps Pipelines.

## Usage

1. Setup your Azure pipeline.
1. Add Bash task to your pipeline.
1. Select inline type.
1. Paste Upload.sh.
1. Change USERNAME, PASSWORD, your-server//path/to/site.
1. Done.
